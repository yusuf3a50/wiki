## Raspberry Pi Models with Built-in WiFi
All wifi cards with raspis have hotspot capability 

### **Raspberry Pi 5 (2023)**
- **WiFi:** 802.11ac dual-band (2.4GHz/5GHz)
- **Chipset:** Broadcom BCM43456

### **Raspberry Pi 4 Model B (2019)**
- **WiFi:** 802.11ac dual-band (2.4GHz/5GHz)
- **Chipset:** Cypress CYW43455

### **Raspberry Pi 3 Model B+ (2018)**
- **WiFi:** 802.11ac dual-band (2.4GHz/5GHz)
- **Chipset:** Cypress CYW43455

### **Raspberry Pi 3 Model B (2016)**
- **WiFi:** 802.11n (2.4GHz only)
- **Chipset:** Broadcom BCM43438

### **Raspberry Pi 3 Model A+ (2018)**
- **WiFi:** 802.11ac dual-band (2.4GHz/5GHz)
- **Chipset:** Cypress CYW43455

### **Raspberry Pi Zero W (2017)**
- **WiFi:** 802.11n (2.4GHz only)
- **Chipset:** Broadcom BCM43438

### **Raspberry Pi Zero 2 W (2021)**
- **WiFi:** 802.11n (2.4GHz only)
- **Chipset:** Cypress CYW43438

### **Raspberry Pi 400 (2020)**
- **WiFi:** 802.11ac dual-band (2.4GHz/5GHz)
- **Chipset:** Cypress CYW43455

## Models WITHOUT Built-in WiFi

The following models require a USB WiFi adapter to create a hotspot:
- Raspberry Pi 1 Model A/B/A+/B+
- Raspberry Pi 2 Model B
- Raspberry Pi Zero (original)
